HTML Basics

What is a tag?

Anything that is enclosed within angle brackets "<>" is a tag.

What is an element?

An element contains a starting tag, content, and an ending tag. Examples: heading elements, paragraph elements, etc.

Using an HTML table, we can store data in a tabular format with rows and columns. The table tag is used to create a table, with tr (table row), td (table data), and th (table header) tags inside.

3 Hrithik 411 ECE Hyderabad

The above table demonstrates the use of different styling properties. The border property value is set to 1, and the table uses the border-collapse property to collapse borders into a single border.

HTML Entities

In HTML, we can't write some characters like the less than (<) and greater than (>) symbols directly.

For these, entities are used: < for less than and > for greater than. Note that there are no spaces in the entities.

Similarly, to display the ampersand (&) symbol, we use the & entity.
