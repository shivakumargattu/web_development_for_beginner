HTML Basics

What is a tag?

Anything that is enclosed within angle brackets "<>" is a tag.

What is an element?

An element contains a starting tag, content, and an ending tag. Examples: heading elements, paragraph elements, etc.

Heading Elements

HTML provides 6 types of heading elements, from h1 to h6. h1 is the highest level and h6 is the lowest level.

Paragraph and Button Elements

The paragraph element is used to write paragraphs.

The button element defines a button.

Lists in HTML

What are the lists in HTML?

In HTML, there are mainly three types of lists:

Unordered List (ul)

An unordered list is a collection of related information that does not follow a specific order. It starts with the <ul> element and by default has bullet points.

Description List (dl)

Definition
The <dl> tag defines a description list. It is used with <dt> (defines terms/names) and <dd> (describes each term/name).

HTML Table

Using an HTML table, we can store data in a tabular format with rows and columns. The table tag is used to create a table, with tr (table row), td (table data), and th (table header) tags inside.
